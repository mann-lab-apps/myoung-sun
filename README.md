# Myoung Sun

Myoung Sun is a mobile-first app for helping a loved one become comfortable using AI in everyday life.

The project is intentionally small and calm: each feature should make the next action clear, reduce fear of mistakes, and support gradual practice.

## Platform

The MVP starts as a PWA so it can be opened on a Samsung Android phone, added to the home screen, and improved quickly without an app-store release cycle.

See [docs/platform.md](docs/platform.md) for the platform decision and Android transition notes.

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run build
```
