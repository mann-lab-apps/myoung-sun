# Low-Anxiety Onboarding

## Goal

The first-run experience should help a hesitant user feel that nothing bad happens when they try.

## Flow

1. Reassure: explain that mistakes are recoverable.
2. Reduce scope: promise one action per screen.
3. Prepare: connect the flow to the first AI prompt practice.

## Interaction Rules

- Show one primary action at a time.
- Keep Back, Skip, and Restart visible in plain language.
- Use encouraging copy without scoring or correction language.
- Keep touch targets at least 52 px tall.
- Prefer short Korean labels that describe the next action.

## Completion Criteria

- The home screen leads directly into onboarding.
- The onboarding can be completed without outside help.
- The final state prepares the user for the first tutorial.
