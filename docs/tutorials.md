# Prompt Tutorials

## Tutorial Shape

Every tutorial follows the same four-step learning pattern:

1. Follow: use a complete example prompt without editing.
2. Fill: change one small phrase in a prepared prompt.
3. Choose: select the desired answer style.
4. Write: try one sentence in the user's own words.

## MVP Topics

- 안내문 쉽게 보기
- 문자 답장 다듬기
- 키오스크 주문 연습

## Design Notes

- Each screen focuses on one action.
- The user can move forward without being graded.
- Prompts are written in everyday Korean and avoid technical AI terms.
- The tutorial data lives in `src/tutorialData.ts` so topics can be edited without changing the app shell.
