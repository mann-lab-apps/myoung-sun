# ChatGPT Guides

## Guide Shape

The app teaches the smallest practical path into ChatGPT through Android WebView:

1. Open `https://chatgpt.com/` inside the app WebView.
2. If login does not work, open ChatGPT in Chrome or another external browser.
3. Log in using the same ChatGPT account method the user already uses.
4. Confirm whether the user is logged in or using ChatGPT without login.
5. Start a new conversation or continue a previous conversation.
6. Copy or organize useful conversation content when needed.

## MVP Guides

- ChatGPT 접속하기
- 로그인하기
- 비로그인/로그인 상태 확인하기
- 새 대화 시작하기
- 이전 대화 이어가기
- 대화 내용 정리/복사하기

## Design Notes

- Keep instructions short enough to read on a phone.
- Avoid abstract prompt-engineering language.
- Prefer direct action labels over complex templates.
- Keep guide text large enough for the Fold7 cover screen.
- Keep the guide UI separate from the ChatGPT WebView page.
- Never automate, scrape, or force-control the ChatGPT page.
- Remind the user to check health, money, or legal decisions with family or a professional.
