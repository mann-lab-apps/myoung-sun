# ChatGPT Guides

## Guide Shape

The app now teaches the smallest practical path into ChatGPT:

1. Open chatgpt.com in Samsung Internet or Chrome.
2. Log in using the same method the user used before.
3. Find the message input at the bottom of the page.
4. Type a short reflection prompt and continue one question at a time.

## MVP Guides

- ChatGPT 들어가기
- 생각정리 시작하기
- 상담하듯 대화하기

## Design Notes

- Keep instructions short enough to read on a phone.
- Avoid abstract prompt-engineering language.
- Prefer copyable first sentences over complex templates.
- Remind the user to check health, money, or legal decisions with family or a professional.
